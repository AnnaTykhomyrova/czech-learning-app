# Czech Learning App 🇨🇿

A small interactive language learning app built with Next.js.

This project was created as a personal tool to help a family member learn Czech in a structured and interactive way.

## Features

- 🧠 Block-based training with progress unlocking
- 🎯 80% pass requirement to unlock next level
- 🎉 Soft success animations and feedback
- ✍️ Custom phrases section
- 🔊 Speech synthesis for Czech pronunciation
- 📝 Edit and delete saved phrases
- 🧩 Multiple choice training mode
- 💾 Local progress saved in localStorage

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- LocalStorage for persistence
- Web Speech API

## Future Improvements

- Progress saving per user
- Pronunciation evaluation
